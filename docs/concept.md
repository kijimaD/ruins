# ゲーム企画書：Ruins（仮称）

## 1. プロジェクト概要

### プロジェクト基本情報
- **タイトル**: Ruins（仮称）
- **ジャンル**: シンボルエンカウント型RPG
- **プラットフォーム**: PC（Windows/Mac/Linux）
- **開発言語**: Go言語
- **想定開発期間**: 6-8ヶ月
- **チーム構成**: 個人開発

### コンセプト
パワプロクンポケットシリーズ風のシンボルエンカウント型ダンジョンRPG。フレーバーテキストを通じて断片的に明かされる世界観と、シンプルでテンポの良い探索・戦闘を楽しむゲーム。

## 2. ゲームシステム

### 基本ゲームループ
```
ディッガー登録 → 遺跡選択 → 探索開始 → 敵シンボルと遭遇 → 戦闘 → アイテム獲得
                ↓
            最深部到達
                ↓
            珠獲得
                ↓
         次の遺跡解放
                ↓
         全遺跡クリア → エンディング
```

### 探索システム
- **自動生成ダンジョン**: 不思議のダンジョン風のランダムマップ生成
- **3つの遺跡**:
  - 山の遺跡: 10階層（初級）→「火の珠」
  - 海底遺跡: 15階層（中級）→「水の珠」
  - 樹海遺跡: 20階層（上級）→「風の珠」
- **特殊部屋**: 各階層に1-2個の固定イベント部屋
  - 見た目が異なる（特別なタイル表示）
  - 環境記録を調査可能
  - 階層ごとに内容は固定、配置はランダム
- **階層ボス**: 各遺跡の最深部に守護者ボスが待ち受ける
- **宝箱**: 各階層にアイテム入り宝箱を配置
- **珠**: 各遺跡の最深部で入手できる神秘的な宝石
- **遺跡間移動**: 不思議な転送装置で各遺跡を行き来

### 戦闘システム
- **ターン制コマンドバトル**: 攻撃・魔法・アイテム・逃走
- **パーティ編成**: 最大4人のパーティ（主人公以外は自動行動）
- **属性**: 物理・火・電・酸・光
- **仲間の特徴**: 各キャラクター固有の戦闘スタイル

### 成長システム
- **レベルアップ**: 経験値を獲得してレベル上昇
- **装備**: 武器・防具・アクセサリの3種類
- **スキル**: レベルに応じて新スキル習得

## 3. 世界観・ストーリーテリング

### 世界観（ゲーム内では明示しない）
- **時代**: 遠い未来の日本
- **舞台**: 地下深くに眠る古代遺跡群
- **技術**: 現代を超えた文明の痕跡
- **謎**: 繰り返される文明の興亡

### ストーリーテリング手法
**環境ストーリーテリング**: 断片的な情報から世界観を推測する探索型物語
- 直接的なストーリーイベントは最小限（開始・珠獲得・終了のみ）
- アイテム説明文、敵図鑑、環境テキストで世界観を表現
- プレイヤーの想像力と考察を促す設計
- 具体的な年代や地名は意図的に曖昧にする

### フレーバーテキスト例

#### 特殊部屋での発見
```
【調査ポイント発見】
「部屋の隅に古びた端末がある。
調べますか？ [はい/いいえ]」

→「はい」選択後

【記録を発見】
「端末にはまだ電源が入っていた。
画面には『第7次文明実験場 - 管理記録』
という文字が表示されている...」

[記録に追加されました]
```

#### アイテム
```
【火の珠】
「赤く脈動する古代の宝珠。触れると微かな温もりを感じる。
裏側に刻まれた文字は現代では解読できない。」

【水の珠】
「深海のような青い輝きを放つ宝珠。見つめていると
遠い記憶が蘇るような錯覚を覚える。」

【風の珠】
「淡い緑の光を帯びた最後の宝珠。手にすると
かすかに震えているのが分かる。」

【エリクサー（試作品）】
「珠から抽出された成分を含む薬品。ラベルには
『被験体番号: 47-MK』と記されている。」

【血染めの写真】
「中年女性の写真。裏面には
『ミキ 47歳 息子へ：頑張って』と書かれている。」

【薬のメモ】
「『MK-47症候群の症状が悪化。
エリクサーの早期投与が必要』医師のサインが読める。」

【古代の通行証】
「転送装置を起動させる金属片。表面には
『第七期管理区域』という刻印がある。」

【換金アイテム】
「古代の銅貨」
「錆びた金属片に刻印が残っている。
『第7次文明実験場 - 職員給与』という文字が見える。」

「データチップ」
「小さな記録媒体。接続すると『被験体行動記録』
というフォルダが大量に並んでいる。」

「機械部品」
「複雑な構造の金属部品。用途は不明だが
表面に『MK-47専用』という刻印がある。」

【装備アイテム】
「実験用防護服」
「白衣に似た防護服。胸のネームプレートには
『研究員 田中』と書かれている。」

「古代の剣」
「刃に青い光が走る不思議な剣。
柄の部分に『第7次実験場警備用』という刻印。」

「エナジーシールド」
「腕に装着する防御装置。起動すると
『生体認証：被験体2847 - 承認』と表示される。」

「治癒のアミュレット」
「温かみのある緑色の石。触れると
『母の愛は永遠に』という文字が浮かび上がる。」
```

#### 敵キャラクター
```
【遺跡の番人】
「古代から遺跡を守り続ける石像。その瞳には
まだ自我が残っているように見える。」

【迷えるディッガー】
「遺跡から帰れなくなった探索者の成れの果て。
首から下げたタグには『娘へ：パパは必ず帰る』と刻まれている。」
```

#### 環境
```
【山の遺跡・入口】
「苔むした石造りの入口。扉の上には
『第7次文明実験場』という文字がうっすらと見える。」

【海底遺跡・転送室】
「青い光に包まれた円形の部屋。中央の装置には
『座標固定：北緯35度 東経139度 深度5000』と表示されている。」

【樹海遺跡・最深部】
「巨大な機械が鎮座する空間。モニターには
『文明進行度: 68% 閾値まで残り32%』という数字が明滅する。」

【共通・休憩所】
「遺跡の所々にある小部屋。壁には前の探索者が残した
『もう何度目だろう』『また同じ夢を見た』という落書きがある。」

【実験記録 - 第2847号】
「被験体は17歳男性。母(47歳)の治療を条件に
志願してきた。感情的反応が強く、実験には最適。」
```

### 最小限のストーリーイベント
1. **ゲーム開始**: 「ディッガー登録番号: 2847 任務: 遺跡探索および『珠』の回収 備考: 母の治療費は組合が負担する」
2. **珠獲得時**: 「珠を回収。必要な素材があと2つ。」
3. **エンディング**: 「全ての珠が揃った。さあ、選べ。」

### 主人公情報の段階的開示
**段階1: システムメッセージ（開始時）**
- 「母の治療」という動機の提示
- 抽象的な情報のみ
- 例：「備考: 母の治療費は組合が負担する」

**段階2: アイテム発見（中盤）**
- 個人的な所持品の発見
- 具体的な名前や関係性の判明
- 例：「血染めの写真」「薬のメモ」で母の名前と病名が判明

**段階3: 記録発見（終盤）**
- 実験記録での真実への言及
- システムの視点からの主人公像
- 例：「実験記録 - 第2847号」で実験の真実が明らかに

**段階4: 環境反応（全体）**
- 特定のアイテムや場所での感情的反応
- プレイヤーの推測を促す表現
- 例：「なぜか胸が締め付けられる」「無性に急がなければ」

## 8. フレーバーテキスト設計指針

### テキスト作成ルール
1. **断片的**: 完全な説明はせず、ヒントのみ提供
2. **具体的**: 抽象的な表現より具体的な描写
3. **感情的**: プレイヤーの感情に訴える要素
4. **一貫性**: 世界観に矛盾しない内容

### カテゴリ別方針
- **アイテム**: 過去の使用者の痕跡、不可解な機能、主人公の個人的な所持品
  - **換金アイテム**: 古代文明の日常品、給与や記録など生活の痕跡
  - **装備アイテム**: 実験場の職員が使用していた道具、警備用装備
  - **消耗品**: 実験用の薬品、被験体用の補給品
  - **特殊アイテム**: 主人公の過去に関連する重要な手がかり
- **敵**: 元は人間だった示唆、哀れみを誘う要素
- **環境**: 文明の痕跡、システムの存在を示唆、主人公の実験記録
- **座標・数値**: 現実を想起させるが確定はさせない
- **感情表現**: 主人公の内面的な反応、推測を促す曖昧な表現

### 内部設定との関連
- 遺跡名は実際の地理を暗示（山→富士、海底→東京湾、樹海→屋久島）
- 座標表示は実在の位置だが、プレイヤーには明示しない
- 「第7次」「文明進行度68%」など、世界観の核心に触れる数値を散りばめる

この企画書は開発の指針として機能し、フレーバーテキストを中心とした環境ストーリーテリングでプレイヤーの想像力を刺激するゲームを目指します。
