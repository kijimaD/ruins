# ゲーム企画書：Ruins（仮称）

## 1. プロジェクト概要

### プロジェクト基本情報
- **タイトル**: Ruins（仮称）
- **ジャンル**: シンボルエンカウント型RPG
- **プラットフォーム**: PC（Windows/Mac/Linux）
- **開発言語**: Go言語
- **想定開発期間**: 6-8ヶ月
- **チーム構成**: 個人開発

### コンセプト
パワプロクンポケットシリーズ風のシンボルエンカウント型ダンジョンRPG。フレーバーテキストを通じて断片的に明かされる世界観と、シンプルでテンポの良い探索・戦闘を楽しむゲーム。

## 2. ゲームシステム

### 基本ゲームループ
```
ディッガー登録 → 遺跡選択 → 探索開始 → 敵シンボルと遭遇 → 戦闘 → アイテム獲得
                ↓
            最深部到達
                ↓
            珠獲得
                ↓
         次の遺跡解放
                ↓
         全遺跡クリア → エンディング
```

### 探索システム
- **シンボルエンカウント**: フィールド上で敵が可視化
- **3つの遺跡**:
  - 山の遺跡: 10階層（初級）→「火の珠」
  - 塔の遺跡: 15階層（中級）→「水の珠」
  - 森の遺跡: 20階層（上級）→「風の珠」
- **階層ボス**: 各遺跡の最深部に守護者ボスが待ち受ける
- **宝箱**: 各階層にアイテム入り宝箱を配置
- **珠**: 各遺跡の最深部で入手できる神秘的な宝石

### 戦闘システム
- **ターン制コマンドバトル**: 攻撃・魔法・アイテム・逃走
- **パーティ編成**: 最大4人のパーティ（主人公以外は自動行動）
- **属性**: 物理・火・電・酸・光
- **仲間の特徴**: 各キャラクター固有の戦闘スタイル

### 成長システム
- **レベルアップ**: 経験値を獲得してレベル上昇
- **装備**: 武器・防具・アクセサリの3種類
- **スキル**: レベルに応じて新スキル習得

## 3. 世界観・ストーリーテリング

### ストーリーテリング手法
**環境ストーリーテリング**: 断片的な情報から世界観を推測する探索型物語
- 直接的なストーリーイベントは最小限（開始・珠獲得・終了のみ）
- アイテム説明文、敵図鑑、環境テキストで世界観を表現
- プレイヤーの想像力と考察を促す設計

### フレーバーテキスト例

#### アイテム
```
【火の珠】
「赤く脈動する古代の宝珠。触れると微かな温もりを感じる。
裏側に刻まれた文字は現代では解読できない。」

【エリクサー（試作品）】
「珠から抽出された成分を含む薬品。ラベルには
『被験体番号: 12-YN』と記されている。」
```

#### 敵キャラクター
```
【遺跡の番人】
「古代から遺跡を守り続ける石像。その瞳には
まだ自我が残っているように見える。」

【迷えるディッガー】
「遺跡から帰れなくなった探索者の成れの果て。
首から下げたタグには『娘へ：パパは必ず帰る』と刻まれている。」
```

#### 環境
```
【山の遺跡・入口】
「苔むした石造りの入口。扉の上には
『第7次文明実験場』という文字がうっすらと見える。」
```

### 最小限のストーリーイベント
1. **ゲーム開始**: 「ディッガー登録番号: 2847 任務: 遺跡探索および『珠』の回収」
2. **珠獲得時**: 「珠を回収。必要な素材があと2つ。」
3. **エンディング**: 「全ての珠が揃った。さあ、選べ。」

### 必須機能（MVP版）
- ターン制戦闘
- 山の遺跡（5階層）のみ実装
- 10種類の敵 + 1体のボス
- 基本的な装備システム
- セーブ・ロード
- 基本フレーバーテキスト（アイテム10種、敵5種、環境5種）
- 珠収集システム

### 完成版追加機能
- 山の遺跡を10階層に拡張
- 塔の遺跡（15階層）追加
- 森の遺跡（20階層）追加

- カットシーン
- 会話システム
- クラフティング
- 拠点建設

## 8. フレーバーテキスト設計指針

### テキスト作成ルール
1. **断片的**: 完全な説明はせず、ヒントのみ提供
2. **具体的**: 抽象的な表現より具体的な描写
3. **感情的**: プレイヤーの感情に訴える要素
4. **一貫性**: 世界観に矛盾しない内容

### カテゴリ別方針
- **アイテム**: 過去の使用者の痕跡、不可解な機能
- **敵**: 元は人間だった示唆、哀れみを誘う要素
- **環境**: 文明の痕跡、システムの存在を示唆

この企画書は開発の指針として機能し、フレーバーテキストを中心とした環境ストーリーテリングでプレイヤーの想像力を刺激するゲームを目指します。
