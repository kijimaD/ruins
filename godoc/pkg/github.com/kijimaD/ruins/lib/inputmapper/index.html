Overview ▾

Package inputmapper はアクション定義とキー入力マッピングを提供する。

このパッケージは、ゲーム内のすべてのアクション（移動、UI操作、アイテム操作など）を ActionIDとして定義し、キーボード入力をActionに変換する機能を提供する。

主な責務: - ActionIDの定義: ゲーム内のすべてのアクションを一元管理 - テスト可能性: キー入力を経由せずにActionを直接実行できる - 拡張性: 新しいアクションの追加が容易
