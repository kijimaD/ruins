Overview ▾

Package consts はゲーム全体で使用される定数と色設定を提供する。

このパッケージには以下が含まれる： - ウィンドウサイズやタイル寸法などのゲーム定数 - UI表示ラベル - アプリケーションバージョン情報 - UI色定数とRGB変換ユーティリティ
